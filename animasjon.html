04_Animasjon

I løpet av temaet Animasjon, fikk vi vite grunnlaget for spilldesign ved å bruke JavaScript og CSS-animasjoner for å skape et interaktivt spill. Formålet var å kombinere teori og konvensjoner innen UI-elementer, figurelementer, bakgrunnsdesign og komposisjonsteknikker for å lage et simpelt men interaktivt spill.

Jeg har utviklet et interaktivt spill fra bunnen av ved hjelp av JavaScript for å implementere spilllogikken og CSS for å redigere designet. Konseptidéen for spillet ble skapt og alle grafiske elementer ble designet i Adobe Illustrator, i tråd med moderne spillkonvensjoner.

Jeg har lært å illustrere skisser og ideer ved hjelp av illustrator, samt grunnprinsippene innenfor javascript og hvordan det henger sammen med html og css for å skape mere avanserte animasjoner og design.
